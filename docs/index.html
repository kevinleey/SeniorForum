General Motivation:

Caretakers need a place to socialize with like-minded individuals, ask questions, learn about events.

Target Users:

Caregivers (Seniors 60+)

Post questions to local community

Browse forums for info on events / helpful tips

Admins

Monitor content

Manage accounts

Core Features:

Users can publish and/or reply to posts about various categorized topics.

Admins can moderate forum content, creating/removing content at will.

Users may connect to specific communities geographically.

Senior-friendly login implementation.

Intuitive UI, using large text and buttons. Should look similar on all devices.

Spell check can assist users that struggle with typing.

Possible Challenges/Obstacles:

Seniors can face challenges in regards to typing/using keyboards.

Minimize click-down windows, double-clicking, etc.

Keep login requirements simple.

Overall design must stay consistent with existing Caregiver Connect sites.

Maintain the mindset of a senior user.

Communicate effectively with sponsor/each other.
