General Motivation:

Caretakers need a place to socialize with like-minded individuals, ask questions, learn about events.

Project Goals:

Sign-in/up: Users should be able to sign in with email/password.

Profile: Users should use this page to display their public information, including name, biography and their post history.

Update Profile: Users should be able to update their public information.

Community: Users should be able to see other users’ posts and comments and interact with them, and also visit other users’ profiles.Users should be able to follow/unfollow other users.

Activity Feed: Users should have a home screen which displays posts from all users, this list is also sortable depending on various criteria such as date posted and trending.

Select Category: Users should be able to track posts based on what category the post is associated with, which users can see the thread count for each category.

Add Post: Users should be able to add a new post, including a title, text, and select categories.

Edit Post: Users should be able to edit features of posts they have created

Comment on Post: Users should be able to add comments on posts.

Upvote Post: Users should be able to upvote posts they like.

Caregiver Connect Information: Users should be able to see information about Caregiver Connect of Alabama, including contact information for them.

Tutorial: Users should be able to see a tutorial for how to use Senior Forum.

Notification: Users should be notified via email when someone comments on their post or when someone follows their account.

Admin Dashboard: Dashboard for non-technical users within the Caregiver Connect of Alabama organization to be able to moderate problematic users.
